This guide lists the QA Wolf commands you can run in the [CLI](../install#understand-the-command-line).

## Use environment variables

You can use [environment variables](environment_variables) when running CLI commands.

For example, run the following to create a test with [`QAW_ATTRIBUTE`](environment_variables#qaw_attribute) set to `my-attribute`:

```bash
QAW_ATTRIBUTE=my-attribute npx qawolf create www.myawesomesite.com
```

## npx qawolf --help

See all commands and options.

## npx qawolf create url [name]

Create a browser [test](../create_a_test) or [script](../create_a_script) on a [Chromium](https://www.chromium.org/Home) browser.

#### Arguments

<ArgumentCreate />

#### Examples

Create a browser test on `www.google.com`, emulating an iPhone 7:

```bash
npx qawolf create --device="iPhone 7" www.google.com
```

Create a browser test on `www.google.com`, saving it at `.qawolf/tests/myTestName.test.js`:

```bash
npx qawolf create www.google.com myTestName
```

Create a browser test on `www.google.com`, saving it at `.myTests/tests/myTestName.test.js`:

```bash
npx qawolf create --rootDir=./myTests www.google.com myTestName
```

Create a brower test on `www.google.com`, loading state data from `.qawolf/state/admin.json`:

```bash
npx qawolf create --statePath=./.qawolf/state/admin.json www.google.com
```

Create a browser script on `www.google.com`:

```bash
npx qawolf create --script google.com
```

## npx qawolf test [name][...options]

Run your test(s) with [Jest](https://jestjs.io/). Tests run on a [Chromium](https://www.chromium.org/Home) browser by default.

#### Arguments

<ArgumentTest />

#### Examples

Run all tests:

```bash
npx qawolf test
```

Run the test saved at `.qawolf/tests/myTestName.test.js`:

```bash
npx qawolf test myTestName
```

Run the test saved at `.qawolf/tests/myTestName.test.js` on [Chromium](https://www.chromium.org/Home), [Firefox](https://www.mozilla.org/en-US/firefox/new), and [WebKit](https://webkit.org):

```bash
npx qawolf test --all-browsers myTestName
```

Run the test saved at `myTests/tests/myTestName.test.js`:

```bash
npx qawolf test --rootDir=./myTests myTestName
```

Use the [`--runInBand` Jest CLI option](https://jestjs.io/docs/en/cli#--runinband) to run tests serially:

```bash
npx qawolf test --runInBand
```
