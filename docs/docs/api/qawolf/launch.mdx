<ArgumentLaunch />

### Returns

<ReturnBrowser />

### Examples

Launch a [Chromium](https://www.chromium.org/Home) browser with the [default options](https://github.com/microsoft/playwright/blob/master/docs/api.md#browsertypelaunchoptions):

```js
const qawolf = require('qawolf');
// ...
const browser = await qawolf.launch();
```

Launch a [Firefox](https://www.mozilla.org/en-US/firefox/new) browser with the [default options](https://github.com/microsoft/playwright/blob/master/docs/api.md#browsertypelaunchoptions):

```js
const qawolf = require('qawolf');
// ...
const firefoxBrowser = await qawolf.launch({ browserName: 'firefox' });
```

Launch a [Firefox](https://www.mozilla.org/en-US/firefox/new) browser, setting the [`slowMo` option](https://github.com/microsoft/playwright/blob/master/docs/api.md#browsertypelaunchoptions) to 50 milliseconds:

```js
const qawolf = require('qawolf');
// ...
const firefoxBrowser = await qawolf.launch({
  browserName: 'firefox',
  slowMo: 50,
});
```
