Convert your browser actions to code. Call `create` in a test or script file to insert new code where it was called.

### Examples

Create new code in an existing file, after calling [`qawolf.register(context)`](register) to register a [Playwright `BrowserContext`](https://github.com/microsoft/playwright/blob/master/docs/api.md#class-browsercontext) instance:

```js
const qawolf = require('qawolf');
// ...
// somewhere earlier in file
qawolf.register(context);

await qawolf.create();
```

Create new code in an existing file, passing a [Playwright `BrowserContext`](https://github.com/microsoft/playwright/blob/master/docs/api.md#class-browsercontext) instance directly:

```js
const qawolf = require('qawolf');
// ...
await qawolf.create({ context });
```

Create new code in an existing file, passing a [Playwright `BrowserContext`](https://github.com/microsoft/playwright/blob/master/docs/api.md#class-browsercontext) instance directly. Use a different selector path than the default:

```js
const qawolf = require('qawolf');
// ...
await qawolf.create({ context, selectorPath: 'my/path/selectors.json' });
```
