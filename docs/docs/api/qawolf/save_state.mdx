<Syntax code="saveState(page, savePath)" />

Save the state of a page (cookies, [`localStorage`](https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage), [`sessionStorage`](https://developer.mozilla.org/en-US/docs/Web/API/Window/sessionStorage)) as [JSON](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/JSON) to the specified file.

This allows you to create tests that load the state before your code runs. See the [handle sign in guide](../../handle_sign_in) to learn more.

### Arguments

<ArgumentSaveState />

### Returns

<ReturnVoid />

### Examples

Save the state of the page, writing or generating code to set the desired state first if needed. For example, you can log in with your UI, or set cookies with Playwright's [`browserContext.setCookies` method](https://github.com/microsoft/playwright/blob/master/docs/api.md#browsercontextsetcookiescookies). See the [handle sign in guide](../../handle_sign_in) for more details.

```js
const qawolf = require('qawolf');
// write or generate code to set desired state
// for example, log in with UI or set cookies
await qawolf.saveState(page, './.qawolf/state/admin.json');
```
